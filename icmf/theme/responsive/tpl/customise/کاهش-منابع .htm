کاهش منابع (HTML، CSS، و جاوا اسکریپت)


این قانون زمانی فعال میشود که PageSpeed Insights تشخیص دهد، کوچک کردن می تواند اندازه منابع را کاهش دهد.

بازنگری

کوچک کردن به فرآیند حذف دادههای غیرالزامی یا زاید، بدون تأثیر گذاشتن بر نحوه پردازش شدن منابع توسط مرورگر گفته میشود - مثلا کد نظرات و فرمت، حذف کدهای استفاده نشده، استفاده از متغیر و نام توابع کوتاهتروغیره. برای اطلاعات بیشتر به بخش Preprocessing & context-specific

توصیه‌ها

شما باید منابع HTML، CSS و جاوا اسکریپت خود را کوچک کنید. شما میتوانید از PageSpeed Insights Chrome Extension برای ایجاد مدل بهینه سازی کد HTML کوتاهتر استفاده کنید. HTML صفحه را تحلیل کرده و به صفحه قانون "Minify HTML" مراجعه کنید. بر روی "See Optimized Content" کلیک کرده تا بتوانید کد HTML کوتاه شده را مشاهده کنید. برای کوچک کردن CSS میتوانید از YUI Compressor و یا cssmin.js استفاده کنید. برای کوچک کردن جاوا اسکریپت میتوانید Closure Compiler، JSMin و یا YUI Compressor را امتحان کنید. شما میتوانید فرآیندی را طراحی کنید که از این ابزار برای کوچک کردن منابع استفاده کرده و فایلهای جدید را با نام جدید در دیرکتوری خاص ذخیره میکند.
